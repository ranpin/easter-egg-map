# 🗺️ 彩蛋地图 Easter Egg Map

> 发现旅途中陌生人留下的温暖与惊喜

基于地理位置的旅行彩蛋 UGC 聚合平台。用户在旅行途中发现或放置的"彩蛋"（如藏在景点的教科书、国家地理杂志、手写信件、盲盒等），通过本平台在地图上标记、记录和分享，形成一个全球范围的"旅行彩蛋寻宝网络"。

## ✨ 在线预览

🔗 [https://06v9evh0zg.ai-app.pub](https://06v9evh0zg.ai-app.pub)

## 🎯 核心功能

- **交互式世界地图**：基于 Mapbox/Leaflet 的全屏地图，标注彩蛋点位
- **彩蛋详情卡片**：点击标记弹出精美卡片，展示图片、标题、描述、来源
- **搜索与筛选**：按城市、分类（书籍/杂志/手写信/盲盒/其他）筛选
- **响应式设计**：适配移动端和桌面端
- **预置示例数据**：覆盖成都、大理、西安、厦门、杭州等热门旅游城市

## 🛠️ 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | React + TypeScript |
| UI 样式 | Tailwind CSS + shadcn/ui 风格 |
| 地图服务 | Mapbox GL JS / Leaflet |
| 后端（规划中） | Node.js + PostgreSQL + PostGIS |
| 部署 | Vercel (前端) + Railway (后端) |

## 💰 盈利模式

| 模式 | 说明 |
|------|------|
| 文旅品牌合作 | 景区/文旅局付费放置官方彩蛋活动 |
| 商家入驻 | 咖啡馆、书店、民宿作为彩蛋点付费标注 |
